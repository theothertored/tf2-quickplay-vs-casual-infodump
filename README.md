# Quickplay vs Casual infodump

Website containing information about TF2's game/match finding, structure and balance.

### [\[ Open site \]](https://theothertored.github.io/tf2-quickplay-vs-casual-infodump)

To export to image, open devtools (Ctrl + Shift + I or F12), right click the element you want to export and select "Capture node screenshot". Zoom the page to export at a higher resolution.
